# Awesome shadcn/ui Admin [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of admin dashboards, CMS platforms, SaaS starters, component extensions, and production apps built with [shadcn/ui](https://ui.shadcn.com).

shadcn/ui is a collection of accessible, customizable React components built on Radix UI and Tailwind CSS. This list focuses on **admin-facing** and **production** use cases.

## Admin Dashboards & Templates

- [shadcn-admin](https://github.com/satnaing/shadcn-admin) - Admin dashboard UI built with shadcn/ui and Vite. Collapsible sidebar, command palette, light/dark mode, RTL support. ★12.4k
- [next-shadcn-dashboard-starter](https://github.com/Kiranism/next-shadcn-dashboard-starter) - Next.js 15 admin dashboard starter with Clerk auth, TanStack Table, charts, and 6+ themes. ★6.6k
- [next-shadcn-admin-dashboard](https://github.com/arhamkhnz/next-shadcn-admin-dashboard) - Next.js 16 + shadcn/ui + Tailwind v4 admin template with customizable theme presets. ★1.1k
- [horizon-shadcn-nextjs-boilerplate](https://github.com/horizon-ui/shadcn-nextjs-boilerplate) - Admin dashboard + ChatGPT AI template with 30+ pre-built components, dark/light mode. ★642

## CMS Platforms

- [UnfoldCMS](https://github.com/hpakdaman/unfoldcms) - **The only CMS with its entire admin built on shadcn/ui.** 210+ admin pages, 51 components, REST API v1, HMAC webhooks, headless mode for Next.js/Astro/SvelteKit/Nuxt. [unfoldcms.com](https://unfoldcms.com)
- [Payload CMS](https://github.com/payloadcms/payload) - Fullstack Next.js CMS framework with TypeScript, headless API, and shadcn/ui-powered admin. ★35k

## SaaS Starters & Boilerplates

- [Open SaaS](https://github.com/wasp-lang/open-saas) - Free React + Node.js SaaS boilerplate with Stripe, auth, email, background jobs, and shadcn/ui. ★14.7k
- [SaaS-Boilerplate](https://github.com/ixartz/SaaS-Boilerplate) - Next.js SaaS starter with Clerk auth, multi-tenancy, RBAC, i18n, Drizzle ORM, and shadcn/ui. ★7.2k
- [Next Enterprise](https://github.com/Blazity/next-enterprise) - Enterprise Next.js boilerplate with Tailwind, TypeScript, Storybook, OpenTelemetry, and CI/CD. ★7.4k

## Component Extensions

- [Payload Components](https://www.payload-components.xyz/) - MIT shadcn-compatible registry and CLI for installing wired Payload CMS blocks into Payload v3 and Next.js projects.
- [shadcn-ui-expansions](https://github.com/hsuanyi-chou/shadcn-ui-expansions) - Extra shadcn/ui components: multiple selector, infinite scroll, datetime picker, floating label input. ★1.9k
- [shadcnspace](https://github.com/shadcnspace/shadcnspace) - 100+ components, 51+ reusable UI blocks, and dashboard kits built with Radix UI and Tailwind. ★617
- [Tremor](https://github.com/tremorlabs/tremor) - 35+ dashboard-focused React components built on Tailwind and Radix UI, copy-paste ready. ★3.5k
- [shadcn-vue](https://github.com/unovue/shadcn-vue) - Vue 3 port of shadcn/ui — same design system, same primitives. ★1.8k
- [AriaKit](https://github.com/ariakit/ariakit) - 25+ unstyled, accessible primitive components for building design systems on top of shadcn/ui. ★8.6k

## Theme & Design Tools

- [tweakcn](https://github.com/jnsahaj/tweakcn) - Visual theme editor for shadcn/ui components. Solve the "every shadcn site looks the same" problem. ★10.1k

## Production Apps

Apps using shadcn/ui in production admin panels — good reference for real-world patterns.

- [Cal.com](https://github.com/cal-com/cal.com) - Open-source Calendly alternative. One of the largest shadcn/ui codebases in production. ★45k
- [Midday](https://github.com/midday-ai/midday) - Invoicing, time tracking, and financial insights for freelancers. Full shadcn/ui admin. ★14.5k
- [Dub](https://github.com/dubinc/dub) - Open-source link attribution platform. 100M+ monthly clicks, shadcn/ui dashboard. ★23.7k
- [Plane](https://github.com/makeplane/plane) - Open-source Jira alternative. Modern admin UI, #1 on GitHub in project management. ★46k
- [OpenStatus](https://github.com/openstatusHQ/openstatus) - Open-source monitoring and status-page platform. Advanced TanStack Table + shadcn/ui patterns. ★5k
- [Cap](https://github.com/cap-so/cap) - Open-source Loom alternative built on Rust, Tauri, and shadcn/ui. ★19k
- [assistant-ui](https://github.com/assistant-ui/assistant-ui) - TypeScript/React library for AI chat UIs with shadcn/ui components. YC-backed. ★9.9k
- [Chatbot UI](https://github.com/mckaywrigley/chatbot-ui) - Open-source AI chat app supporting 80+ models. shadcn/ui React frontend + Supabase. ★33.3k

## Related Libraries

Core dependencies used across shadcn/ui projects.

- [shadcn/ui](https://github.com/shadcn-ui/ui) - The component library itself. Copy-paste components built on Radix UI + Tailwind. ★112k
- [Radix UI Primitives](https://github.com/radix-ui/primitives) - Low-level accessible primitives that power shadcn/ui. ★19k
- [Lucide Icons](https://github.com/lucide-icons/lucide) - 1600+ consistent SVG icons used throughout the shadcn/ui ecosystem. ★23k
- [Recharts](https://github.com/recharts/recharts) - React + D3 chart library used in shadcn/ui chart components. ★27k
- [TanStack Table](https://github.com/TanStack/table) - Headless table library used for data tables in shadcn/ui admin patterns. ★26k
